Notes
  • The default unit of measure for weight is kilograms. Please verify that the correct unit of measure has been selected.
  • This formula has lower precision in neonates and infants, compared with older age groups, and if height cannot be accurately measured for any reason. [2] When dosing sensitive medications (eg, chemotherapy), some protocols use BSA-based dosing for children but use other approaches for dosing in infants (eg, weight-based).


Equations used

BSA = SquareRoot(Height * Weight / 3600)




References
  1. Mosteller RD. Simplified calculation of body surface area. NEJM 1987; 317:1098.
