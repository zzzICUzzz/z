Equations used

Log_odds = (0.0287 * (Age - 62)) + Sex + Family_History_Lung_Ca + Emphysema - (5.3854 * ((Nodule_size/10) -0.5 - 1.58113883)) + Nodule_type + Nodule_Upper_Lung - (0.0824 * (Nodule_count - 4)) + Spiculation - 6.7892
Cancer_probability = 100 * (e (Log_odds) / (1 + e (Log_odds) ))




Equation parameters, such as Sex , have 2 or more discrete values that may be used in the calculation. The numbers in the parentheses, eg, (0.6011), represent the values that will be used.
