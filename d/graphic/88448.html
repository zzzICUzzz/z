Glomerular lesions in schistosomiasis
A-D are stained with H&E. F is stained with Masson's trichrome.
(A) Class I: Axial mesangial proliferation.
(B) Class II: Salmonella-associated exudative glomerulonephritis (GN).
(C) Class III: Membranoproliferative (mesangiocapillary) GN.
(D) Class IV: Focal segmental sclerosis.
(E) Class V: Amyloidosis.
(F) Class VI: HCV-associated mixed lesion including mesangial proliferation amidst amyloid and cryoglobulin deposits.
HCV: hepatitis C virus.
