Parkland Formula nomogram for adult burns
Instructions (summary):
  1. Place a straight edge so that it intersects the correct values for both BWt (kg or stones) and TBSA (%), and read VTBI (L) per period.
  2. Place a straight edge so that it intersects the correct values for both VTBI (L), and the time remaining in the FIRST PERIOD (h) from time of the burn injury.
    Give the VTBI (L) as Hartmann's solution at the rate indicated where the straight edge intersects the RATE (mL/h or drops/min) scale until the end of the First Period.
  3. Then: Place a straight edge so that it intersects the correct values for both VTBI (L) and the cross marked "SECOND PERIOD".
    Give the VTBI (L) as Hartmann's solution at the rate indicated where the straight edge intersects the RATE (mL/h or drops/min) scale until the end of the Second Period (16 h).
  4. If additional resuscitation fluid has been given to the patient prior to admission, subtract this volume from the VTBI (L) in step 2); but VTBI (L) in step 3) remains unchanged.
